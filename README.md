## Getting Started

### Prerequisites
- Node.js (v20+)
- npm

### Installation
```bash
npm install
```

### Development
```bash
npm run electron:dev
```

### Testing
```bash
npm test
npm run test:coverage
```

### Manifestation (Build)
To generate the production-ready `.exe`:
```bash
npm run dist
```
